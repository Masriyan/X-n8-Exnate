# X-n8-Exnate
X-n8 (Exnate) is an Agentic SOC Playbook collection bridging n8n’s AI agility with Palo Alto XSOAR’s orchestration. Covering 250+ use cases—from Cloud to IAM—it uses AI Agents for dynamic SIEM alert analysis and automated remediation. Aligning with MITRE ATT&amp;CK, X-n8 transforms manual workflows into a scalable, autonomous defense ecosystem.
